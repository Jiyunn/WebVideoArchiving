# 웹 동영상 컨텐츠 아카이빙 및 공유 앱 만들기
## 프로젝트 개요
* 웹의 동영상 컨텐츠의 URL을 아카이빙 하여 손쉽게 다시 볼 수 있다.
* 아키이빙 한 동영상 컨텐츠를 다른 사람에게 공유 할 수 있다.
* 아키이빙 한 동영상 컨텐츠의 특정 구간을 지정하여 쉽게 다시 볼 수 있다.
* 아카이빙 한 동영상 컨텐츠의 특정 구간을 지정하여 다른 사람에게 공유 할 수 있다.
